ornl.elision.core

Timeable

trait Timeable extends AnyRef

Specify that it is possible to enable and disable printing the execution time of each atom that is parsed. To use this you must implement the reportElapsed method that reports the elapsed time, and make sure that the startTimer and stopTimer methods are called at the appropriate places.

Do not use reportElapsed directly. Implement it, and then invoke showElapsed at the appropriate place in your code. The showElapsed method does nothing if timing is disabled, and invokes reportElapsed if it is.

Even when timing is disabled, the elapsed time information is still kept by startTimer and stopTimer, and can be checked by getLastTimeMillis, getLastTime, or getLastTimeString.
