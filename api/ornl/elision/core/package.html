package core

The core classes and definitions that make up the Elision runtime.

Design Goals

  • Every atom that can be programmatically created can be written using the toParseString, and the result can be parsed by AtomParser.
  • Every atom that can be programmatically created has a toString method that generates valid Scala code to reproduce the atom.
  • Avoid global data and singletons.
  • Simple API.
