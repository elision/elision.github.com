  19. var current : RWTreeNode

    Current is used to store a reference to a RWTreeNode representing an atom currently being rewritten.

    Current is used to store a reference to a RWTreeNode representing an atom currently being rewritten. Before calling rewrite on an atom, you should first create a RWTreeNode for that atom and then set current to that new node. Then you can retrieve current for processing inside that atom's rewrite method. Maintaining this reference makes it so that the parameter lists for rewrite methods don't have to be rewritten to accomodate passing down RWTreeNodes to construct the rewrite tree structure during the rewrite process.
