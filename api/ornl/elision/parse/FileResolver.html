  1. new FileResolver(usePath: Boolean = true, useClassPath: Boolean = true, path: Option[String] = scala.None)

    usePath

    If true, use the search path. This can be provided via the setPath methods, or from the ELISION_PATH environment variable, if set. In case neither is set, the path . is used. Default is true.

    useClassPath

    If true, use the class path to search for files. This is always used after attempting to use the search path. Default is true.

    path

    Optional search path. Default is None, and the path is initialized from ELISION_PATH as described above.

  11. def find(name: String): Option[(InputStream, String)]

    Locate the specified file in the allowed search paths.

    Locate the specified file in the allowed search paths.

    If the provided name is an absolute path, no search is performed. Otherwise the search path is used first (if enabled) and then the class path is used (if enabled). If the file is found, a reader is created and returned. Otherwise None is returned.

    name

    The file name, possibly including leading path.

    returns

    None if not found, or a reader to read the file and the path of the file's parent directory.
