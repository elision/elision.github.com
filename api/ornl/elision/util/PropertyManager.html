PropertyManager

trait PropertyManager extends AnyRef

Manage a collection of properties.

To use this trait simply inherit it and implement the clazzes abstract value to the list of classes that should be allowed. If there are special values that need a unique representation when written, override the writeProperty value.

  1. abstract val clazzes: Set[_ <: Class[_]]

    The classes that are supported by this property manager.

    The classes that are supported by this property manager. Give this as a set of class objects. This is easily done. The following is an example that allows storing (and retrieving) Ints, Strings, and Floats.

    val clazzes = Set(classOf[Int], classOf[String], classOf[Float])

    The property values x must be assignable to a variable of one of the given types. That is, there must be a class C in this set such that

    C.isAssignableFrom(x)

  31. def writeProperty(value: Any): String

    Write a single property.

    Write a single property. This is used by writeProperties to write the value of each property. If you have a property that requires special handling or representation, override this. Use super.writeProperty for values you do not wish to handle.

    Alternately you can just override this method in the derived class for the values you want to catch.

    For example:

    def writeProperty(value: Int) = "[+value+"]"

    This will write Int values with square brackets, and will defer to the base class for all other values.

    By default this method uses the toString method for the value.

    value

    The value to write.

    returns

    The string value to write.
