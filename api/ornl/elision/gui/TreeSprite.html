TreeSprite

class TreeSprite extends Sprite

Contrary to what the name suggests, this class is not some sort of mystical nature spirit. Rather, it represents the rewrite tree structure for an Elision term as a renderable sprite.

  12. def countLeaves (node: NodeSprite): Int

    Recursively counts the number of leaf nodes among node's descendants.

    Recursively counts the number of leaf nodes among node's descendants. A node is counted as a leaf node as far as this method is concerned if it is decompressed and either it has no children or all its children are compressed. This method recursively computes the number of leaves for all decompressed nodes in the tree and stores them inside the nodes for easy reuse.

    node

    The node whose leaf descendants we are currently counting

    returns

    The number of node's leaf descendants
