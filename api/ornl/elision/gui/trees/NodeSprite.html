ornl.elision.gui.trees

NodeSprite

class NodeSprite extends Sprite

A sprite used to represent a rewritten term node in a TreeSprite. The x,y coordinates for NodeSprites are relative to their parent's x,y coordinates. In the case of a TreeSprite's root node, this should be 0,0.
