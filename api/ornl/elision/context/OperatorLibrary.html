OperatorLibrary

class OperatorLibrary extends Fickle with Mutable

An operator library holds information about the known operators. Operators are managed by name, and two operators in the same library cannot have the same name (but there can be multiple libraries).

Native Operators

Handlers for native operators come here to register themselves. Pass a closure of the proper form (see register).
