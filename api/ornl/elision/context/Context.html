ornl.elision.context

Context

class Context extends Fickle with Mutable with Cache

A context provides access to operator libraries and rules, along with the global set of bindings in force at any time.

Additionally the context maintains a cache for use during runtime. This cache is reflected through an Executor, but is actually maintained here.

This class is likely to change.

Use

In general it is not necessary to make an instance; one is typically provided by a higher-level (semantically) class.

This class provides for management of four things:

  • A set of Bindings.
  • An instance of ornl.elision.core.OperatorLibrary.
  • Rulesets.
  • "Automatic" rewriting of atoms using rules.

  15. def fetchAs[TYPE](key: String, default: TYPE)(implicit mTYPE: Manifest[TYPE]): TYPE

    Provide typed access to the content of the cache.

    Provide typed access to the content of the cache. This is intended for use by native operators. To avoid conflicts, name your cache entries starting with your operator name. This causes a performance hit, so in general avoid using the cache. Find somewhere else to shove your data!

    If the key is present, but of the wrong type, an exception is thrown. This is a CacheException.

    key

    The key for the item to retrieve.

    default

    The value to return if the specified key is not present. If this is returned, it is also stored, so be sure to type it correctly.

    returns

    The requested value, or the default value.
