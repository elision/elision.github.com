It is *under construction*, so set your expectations accordingly. The rewriter's initial feature set is stable, but the code itself is still undergoing testing. To build Elision you must have [Scala][scala] and [Apache Ant][a nt] installed. Elision is built and tested on Linux and Mac OS X using Java 1.7 and Scala 2.10.1. To use the runnable jar (for Scala), execu te it with the scala command:<br/> `scala elision.jar` To use the runnable jar (for Java), execute it with the java command:<br/> `java -jar elision-all.jar` ## Acknowledgments - Thanks to [Headway Software][headway] for providing licenses for structure101. - Elision contains [JLine][jline], a library for reading from a prompt. - This page was created using [Markdown][markdown] and [Strapdown][strapdown].
